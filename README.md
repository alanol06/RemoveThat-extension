# Remove That!

Extension that lets you delete any element on a webpage with one click.

![Chromium Extension](https://img.shields.io/badge/Chrome-Extension-brightgreen)

- üéØ **One-Click Removal** ‚Äî Hover and click to instantly delete any element from the page  
- üîç **Live Highlighting** ‚Äî Bright red outline and subtle background show exactly what will be removed  
- ‚ú® **Particle Explosion** ‚Äî Satisfying burst of colorful particles on every deletion  
- ‚ö° **Instant Toggle** ‚Äî Enable/disable mode with a single click on the toolbar icon  
- üõë **Quick Exit** ‚Äî Press **Esc** anytime to leave removal mode instantly  
- üí® **Super Lightweight** ‚Äî Minimal resource usage with a clean, fast, and distraction-free experience  

## Installation

**Google Chrome / Chromium-based browsers:**
1. Open `chrome://extensions/`
2. Enable **Developer mode** (top right corner)
3. Click **Load unpacked**
4. Select the `RemoveThat` folder

**Microsoft Edge:**
1. Open `edge://extensions/`
2. Click **Load unpacked**
3. Select the `RemoveThat` folder

## Usage
1. Click the extension icon in your browser toolbar to activate removal mode  
2. Hover over any element ‚Äî it will get a red highlight  
3. Click the element ‚Äî it disappears with a particle effect  
4. Press **Esc** or click the icon again to exit the mode  

Done ‚Äî the page is cleaned exactly the way you want.

## Notes
- Deletions are temporary (reset on page refresh)  
- `<body>` and `<html>` elements are protected from removal
